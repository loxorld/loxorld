## 📌 Featured projects

### 🔹 [Portfolio API](https://github.com/loxorld/portfolio-api)
Backend API for my personal portfolio, built with **Spring Boot, Spring Data JPA, Spring Security, and PostgreSQL**.  
Includes public project endpoints and protected admin endpoints for content management.

### 🔹 [Volve a Casa](https://github.com/loxorld/volve-a-casa)
Full stack web application for **lost pets, sightings, and community coordination**, built with **Angular, Spring MVC, Hibernate, JWT, and MySQL**.

### 🔹 [Es Estafa?](https://github.com/loxorld/es-estafa)
Web app built with **Next.js and TypeScript** to analyze suspicious messages and links using **local heuristics plus optional AI support**.

### 🔹 [StockManager WPF](https://github.com/loxorld/stock-manager-wpf)
Desktop inventory management application built with **WPF and SQLite**, focused on SKU management, stock movements, and sales metrics.

### 🔹 [PegLoxGame](https://github.com/loxorld/PegLoxGame)
Android game prototype developed in **Unity**, where I explore gameplay systems, architecture, and technical problem solving from a different angle.
